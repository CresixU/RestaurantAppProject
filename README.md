# RestaurantAppProject
A console application that imitates the behavior of a popular fast food restaurant, made in its own way

## Features
- Readable and colorful menu
- Login and singup system
- Saving data to file


## Libraries
- Spectre.Console.Cli
- SimpleHashing.NET
- Newtonsoft.Json
